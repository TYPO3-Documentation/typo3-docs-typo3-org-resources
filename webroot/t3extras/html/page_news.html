Tonight the TYPO3 community released the second and last release candidate of TYPO3 4.5. This represents a final rehearsal for the final release of tomorrow.

Development progress

The release plan for 4.5, the schedule and the planned projects are detailedin the project wiki on forge. For details about this particular release and the current progress, please read up on:
http://forge.typo3.org/projects/typo3v45-projects/wiki/TYPO3_450rc2

Downloads

We also have an introductionpackage for rc2, which allows testing the new features right away on a fresh installation. Experienced users can of course opt to upgrade an existing installation to the new 4.5.0rc2 version, but keeping in mind that minor changes will still happen until the final release tomorrow.

You can download the packages from:
http://typo3.org/download/packages/
